🚗 CarLink Marketplace

CarLink is a simple and user-friendly car marketplace app that allows users to list cars for sale or rent and connect directly with buyers via WhatsApp.

Features

- Add car (name, price, image)
- Save data locally
- Delete listings
- WhatsApp contact
- Clean interface
- Basic login system

How It Works

1. Register or login
2. Add your car details
3. Upload image
4. Car appears in listings
5. Buyers contact via WhatsApp

Tech Used

HTML, CSS, JavaScript, LocalStorage, WebView

Future Updates

- Online database (Firebase)
- Real user accounts
- Search & filters
- Payments
